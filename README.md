# KNN Classification Project

## Objective
The aim of this project is to implement the K-Nearest Neighbors (KNN) algorithm for classification using the Iris dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Steps Performed
1. Loaded the Iris dataset.
2. Converted data into a DataFrame.
3. Split data into training and testing sets.
4. Normalized features using StandardScaler.
5. Trained a KNN classifier.
6. Tested different values of K.
7. Evaluated model performance using accuracy score.
8. Generated confusion matrix and classification report.
9. Visualized accuracy for different K values.
10. Plotted decision boundaries.

## Results
The KNN model achieved high accuracy on the Iris dataset and successfully classified flower species.

## Conclusion
KNN is a simple and effective machine learning algorithm. Choosing the correct K value and normalizing data improves performance.
